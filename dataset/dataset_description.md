# Dataset Description

## Source
Dataset provided by Praxis 2.0 organizers.

## Description
The dataset contains student demographic, academic, and behavioral attributes used to analyze learning patterns.

## Key Features
- Demographics: age, sex, address
- Academic behavior: studytime, failures, absences
- Engagement indicators: activities, internet, higher education
- Performance scores: G1, G2, G3

## Potential Targets
- Final performance (G3)
- Learning risk categories
- Engagement levels
